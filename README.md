# Notes Not Scandal

A personal collection of clinical and academic notes by Ines Tan — infectious disease, hematology, and more — published with MkDocs from an Obsidian vault. The site name is a tongue-in-cheek nod to Cate Blanchett's *Notes on a Scandal*.

Live site: <https://didiowen.github.io/notes-not-scandal/>

Ines Tan 的個人臨床與學術筆記合集，涵蓋感染症、血液腫瘤、專師考試解題等，從 Obsidian 筆記庫透過 MkDocs 發布。網站名取自凱特·布蘭琪電影《Notes on a Scandal》（中譯《醜聞筆記》）。
