# 醫學會行事曆 (ICS 訂閱)

以下 ICS 檔由 [society-calendar](https://github.com/didiowen/society-calendar) 中的
skill 自動從各學會官網抓取活動產生，依下列規則過濾後手動更新：

- **台南** 一律收錄
- **高雄 / 嘉義**：學分數 > 2
- **其他地區**：學分數 > 3

## 訂閱方式

Google Calendar → 左側「其他日曆」旁的 ➕ → **透過網址新增日曆** → 貼上下列任一連結。
（Apple Calendar / Outlook 也都支援同一網址。）
