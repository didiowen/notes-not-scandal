# Non-typhoid Salmonella Vascular Infection  
## Chen-NTSVI score [@Chen2012simplescoring]  
  
| RIsk factors              | Point |  
| ------------------------- | ----- |  
| male sex                  | +1    |  
| hypertension              | +1    |  
| coronary arterial disease | +1    |  
| serogroup C1              | +1    |  
| immunosuppressive therapy | -1    |  
| malignancy                | -1    |  
  
 ≥ +1 → arrange CT  
 Se: 95.0%  
 Sp: 45.3%  
 AUC: 0.83 (0.78-0.89)
